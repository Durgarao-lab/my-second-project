# Mechanic Booking System
A full-stack Mechanic Service Booking Web Application built using Flask (Python) where customers can book nearby mechanics, track service status, and submit reviews.

# Customer
- Sign up & login
- View nearby available mechanics
- Book a mechanic for different services
- Track booking status (Pending / Accepted / Completed)
- Cancel booking
- Give ‚≠ê ratings & reviews after service
- Edit profile & upload profile photo

# Mechanic
- Login & dashboard
- Toggle availability (Online / Offline)
- Accept or reject customer bookings
- View assigned jobs
- Track earnings
- View customer reviews & rating
